# best-admin

### 介绍
该项目主要是一个用于开发的基础框架，以供给自己快速开发项目

* [后台项目地址](https://github.com/shanzhaozhen/best-server)
* [前端项目地址](https://github.com/shanzhaozhen/best-client)
* [Demo站点](http://best.loogoos.tk)
